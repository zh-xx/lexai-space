# 法律AI工作台

一个集成了多个法律AI工具的统一工作平台，为律师和法务人员提供智能化的法律服务。

## 🚀 功能特性

### 已实现功能
- **合同审核工具**：智能分析合同风险，提供详细的风险评估报告
  - 支持.docx文档上传和解析
  - 多维度风险分析（主体、基础条款、业务条款、法律条款）
  - 可视化图表展示分析结果
  - PDF报告导出功能
  - 实时进度显示和错误处理
- **合同生成工具**：基础框架已搭建完成，支持合同模板生成
- **合同解析工具**：智能解析合同内容，提取关键信息
  - 支持文件上传和文本输入
  - 拖拽上传功能
  - 八项关键信息提取
  - 卡片式结果展示
- **统一工作台**：侧边栏导航，支持多工具切换
- **响应式设计**：适配桌面端和移动端
- **搜索功能**：快速查找工具
- **最近使用**：记录最近使用的工具

### 开发中功能
- **合同生成工具**：完善智能合同生成功能
- **更多法律工具**：起诉状、答辩状等文书生成
- **合同解析工具**：集成Coze工作流API

## 🛠️ 技术架构

### 前端技术
- **HTML5 + CSS3**：现代化界面设计
- **JavaScript ES6+**：模块化开发
- **Font Awesome**：图标库
- **Google Fonts**：字体优化

### 核心库
- **mammoth.js**：Word文档(.docx)解析
- **ECharts**：数据可视化和图表生成
- **html2pdf.js**：PDF报告导出
- **marked.js**：Markdown内容渲染
- **mermaid.js**：流程图生成
- **JSZip**：文件压缩处理

### 设计特点
- **侧边栏导航**：清晰的功能分类
- **iframe嵌入**：工具独立运行
- **平滑动画**：良好的用户体验
- **移动端适配**：响应式设计
- **90%缩放适配**：模拟浏览器缩放效果

## 🚀 快速开始

### 1. 启动项目
```bash
# 进入项目目录
cd contract_review

# 启动本地服务器
python3 -m http.server 8000

# 或者使用Node.js
npx http-server -p 8000
```

### 2. 访问应用
打开浏览器访问：`http://localhost:8000/main.html`

### 3. 使用说明
1. **工具切换**：点击侧边栏的工具图标切换不同工具
2. **搜索工具**：使用搜索框快速查找工具
3. **最近使用**：查看最近使用的工具
4. **移动端**：点击左上角菜单按钮展开侧边栏

## 🔧 开发指南

### 添加新工具
1. 在 `tools/` 目录下创建新的工具页面
2. 在 `main.html` 的侧边栏中添加工具入口
3. 在 `ToolManager` 类中注册新工具
4. 实现工具与父框架的通信

### 工具页面模板
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>工具名称</title>
    <!-- 引入样式和字体 -->
</head>
<body>
    <!-- 工具内容 -->
    
    <script>
        // 与父框架通信
        window.addEventListener('message', (event) => {
            if (event.data.type === 'getStatus') {
                window.parent.postMessage({
                    type: 'status',
                    tool: 'tool-id',
                    status: 'available'
                }, '*');
            }
        });
    </script>
</body>
</html>
```
